## 1. 프로젝트 개요 및 비즈니스 토폴로지

본 시스템은 대형 호출 플랫폼(카카오, 우버 등)에 종속되지 않는 개인 및 프리미엄 대형 택시 자영업 주체 전용의 독창적인 가치 순환 체계를 기술적으로 증명하고 구현합니다. 

기사의 운전 데이터(비동기 주행 궤적 및 매출 결산 데이터)를 플랫폼에 공급하고, 플랫폼은 지능형 핫존 예측 맥락 피드백과 세무 정산 자동화 가치를 역으로 서빙하여 상호 윈윈(Win-Win)하는 B2B2C 결합 생태계를 조기 정착시킵니다.

### 👥 조직 및 역할 (Roles)
*   **PM / 기획 Gem**
    *   요구사항의 예외 규칙을 정의하고 Supabase 스키마 관계형 매핑 구조 설계.
*   **디자인 / 퍼블 Gem**
    *   Stitch 자산 기반 고유 컬러 파레트 및 마이크로 인터랙션 UI 토큰 수립.
*   **개발 / 운영 Gem**
    *   LangGraph 상태 머신 모델 설계, 결함 격리 가드레일 및 비용 최적화 검증.

## 2. 세부 기능 요구사항 (Core Functional Specifications)

### 📌 REQ-01: GILLOG 기반 출근 루틴 및 콘텐츠 인게이지먼트
*   **루틴 송출**: 매일 오전 07:00 출근 타임라인에 맞춰 기사의 띠/사주 데이터 기반 행운 카드 오브젝트 및 당일 아침 최적 드라이빙 코스 추천 레이아웃을 브라우저 푸시로 강제 송출합니다.
*   **콘텐츠 큐레이션**: '택린이 성장계획서'의 핵심 비즈니스 모델인 '양도양수 리얼 후기', '수입 공개 리포트', '초보 생존 가이드' 콘텐츠 아티팩트를 RSS 및 Open API 형태로 메인 대시보드 큐레이션 레이어에 포워딩하여 정서적 락인을 형성합니다.

### 📌 REQ-02: G-PAN 지능형 오디오 관제 엔진 (Zero-Touch TTS)
*   **맥락 데이터 수집**: 기상청 실시간 날씨, 서울시 도로별 돌발 트래픽, 주요 공항 연착 및 지하철 지연 파싱 맥락 데이터를 실시간 비동기 수집하여 유망 핫존 구역 추정 알고리즘을 가동합니다.
*   **오디오 피드백**: 텍스트 형태로 컴파일된 예측 데이터를 라디오 방송 형태의 비동기 오디오 스트리밍 API로 변환하여 주행 중 수동 조작 없이 전방 주시 중심의 무인 오디오 가이드로 송출합니다.

### 📌 REQ-03: Tax Autopilot 기반 경영 자율비행 코어
*   **매출/매입 데이터 연동**: CODEF 및 쿠콘 금융 대행사 API 게이트웨이를 연동하여 여신금융협회 카드 매출 데이터 스크래핑 파이프라인을 구축합니다.
*   **세무 자율 처리**: 차량 제조사 제휴 매입액, 타이어 유통 네트워크, 대형 정비 인프라 실시간 스토어 정산 데이터를 국세청 홈택스 API 신고 규격과 융합하여 원클릭 부가세 환급 자율비행 로직을 실행하여 복잡한 행정 소요를 제로화합니다.

### 📌 REQ-04: Roadboarder 커뮤니티 및 리더보드
*   **매출 리더보드**: 당일 운행 매출 인증 탑 보더들의 주행 패턴, 시간대별 수익 스키마 데이터를 테이블 컴포넌트로 실시간 공유하여 선의의 경쟁을 유도합니다.
*   **기사 광장 피드**: 장년층 전용 가독성(폰트 크기 130% 상향)을 반영한 심플 UI 기반 커뮤니티 피드를 가동하여 정서적 피로 해소 네트워크를 구축합니다.

## 3. 정보 아키텍처 및 메뉴 구성 (Information Architecture)

장년층 최적화 극강의 심플 UI 디자인 규격을 적용하여 프론트엔드(`fo/src/pages`) 계층 구조를 선언합니다.

*   **MENU 01: 오늘의 루틴 (GILLOG 홈)**
    *   `[Component]` 사주 기반 행운 카드 컨테이너
    *   `[Component]` 아침 핵심 가이드 코스 브리핑 배너
    *   `[Component]` 택린이 초보 생존 가이드 미디어 피드
*   **MENU 02: G-PAN 레이더 (실시간 관제)**
    *   `[Component]` Zero-Touch 전방 주행 오디오 토글 스위치 (ON/OFF)
    *   `[Component]` 맥락 데이터 통합 트래픽 가시성 배너
    *   `[Component]` 실시간 택시 대기열 시각화 서비스 뷰
*   **MENU 03: 로드보더 (커뮤니티)**
    *   `[Table]` 전국 매출 Top 3 리더보드 컴포넌트
    *   `[Component]` 기사 광장 실시간 타임라인 스레드
*   **MENU 04: 오토파일럿 (경영/세무)**
    *   `[Chart]` 이번 달 누적 수입 및 고정비 지출 분석 대시보드
    *   `[Button]` 원클릭 홈택스 세무 자동 정산 신청 트리거
    *   `[Component]` B2B 정비/타이어 실속 제휴 몰 커머셜 가교

## 4. 기술 제약 조건 및 글로벌 보안 가드레일 (Technical Constraints)

### ⚙️ LangGraph 단일 책임 노드(SRP) 아키텍처
> [!IMPORTANT]
> **에이전트 제어 및 비용 절감**
> *   **노드 구성**: `시작` ➔ `트래픽 스크래핑` ➔ `임베딩` ➔ `핫존 검색` ➔ `오디오 요약` ➔ `TTS 최종 송출` 순으로 독립 노드를 분리하여 에이전트 상태 전이를 제어합니다.
> *   **결함 격리**: 런타임 예외 또는 무한 루프 징후 감지 시 제어를 `State.Error` 인터페이스로 즉시 격리 이관하고 파이프라인을 즉시 중단 시킴으로써 불필요한 인프라 및 LLM API 비용 폭증을 원천 차단(Prevent LLM Cost)합니다.

### 🛡️ REQ-05: Risk Shield 컴플라이언스 샌드박스
> [!WARNING]
> **플랫폼 제재 우회 및 독립성 유지**
> 거대 플랫폼의 다중 호출 제재 규정을 우회하기 위해, 타사 호출앱 소스코드 및 프로세스에 직접 개입하는 행위를 절대 엄금합니다. 오직 OS 레벨 시스템 알림 인프라 가로채기 헬퍼 및 외부 맥락 수집을 위한 비동기 백그라운드 데이터 패칭 구조로만 설계하여 논리적 방벽과 컴플라이언스 안전 우회로를 100% 보장합니다.

### 🔒 엔터프라이즈 전역 보안 Baseline (Global Policy)
1.  **Zod 스키마 검증**
    *   모든 인바운드/아웃바운드 API 접점 및 핀테크/공공 데이터 파싱 밸류를 `Zod` 스키마 런타임 검사로 강제하여 데이터 오염 및 타입 브레이크를 원천 차단합니다.
2.  **DOMPurify 기반 XSS 필터링**
    *   `dangerouslySetInnerHTML` 호출을 전면 차단하며, 에이전트 요약 결과 및 마크다운 렌더링 시 반드시 `DOMPurify.sanitize` 독립 필터 파이프라인을 통과해야 합니다.
3.  **WHATWG URL 프로토콜 준수**
    *   외부 제휴 커머스 및 외부 쇼핑몰 아웃링크 이동 시 `WHATWG URL` 표준 모듈 필터링 계층을 배치하여 `http:`, `https:` 프로토콜 화이트리스트 검증을 통과한 자원만 서빙합니다.
4.  **외부 아웃링크 안전 격리 정책**
    *   써드파티 외부 사이트로 포워딩되는 모든 링크 오브젝트에는 브라우저 스레드 제어권 탈취 예방을 위해 `target="_blank" rel="noopener noreferrer"` 속성 삽입을 강제 집행합니다.
