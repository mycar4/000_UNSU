# UNSU 프론트엔드 실시간 Mock API 및 인터랙티브 피쳐 개발 요약

프론트엔드 우선 개발 정책에 의거하여, 단순한 시각 목업 수준을 넘어 실제 기사가 조작할 수 있는 **동적 상태 전이(State Transition) 및 브라우저 네이티브 하드웨어 API 연동**을 구현했습니다.

---

## 1. 구현된 핵심 인터랙티브 기능

### ① 기사 온보딩 마스터 프로필 연동 ([OnboardingPage](file:///d:/000_UNSU/fo/src/pages/OnboardingPage.tsx))
*   입력된 생년월일, 출생 시간, 사업자 종류, 홈택스 ID 정보를 Zod로 정밀 밸리데이션 검사한 뒤 `localStorage`에 `driverProfile` 키로 암호화 적재합니다.
*   이 프로필 정보는 이후 **오늘의 루틴(Gillog)** 및 **오토파일럿(정산)** 도메인의 비즈니스 로직 연산 핵심 원천 데이터로 작용합니다.

### ② 사주 만세력 알고리즘 및 온보딩 유도 ([GillogPage](file:///d:/000_UNSU/fo/src/pages/GillogPage.tsx))
*   **미등록 예외 처리**: 프로필이 등록되지 않은 기사가 진입 시, 상단에 **"프로필 미등록 알림 배너"**를 골드 뱃지와 함께 노출하며 클릭 시 즉시 온보딩 페이지로 다이렉트 안내합니다.
*   **결정론적 일진 연산**: 기사 생일과 당일 날짜 문자열을 결합한 유니크 해시 수식을 빌드하여, 외부 유료 사주 API 호출 없이 기사 생일마다 매일 다르게 매핑되는 **"사주 기반 오늘의 행운 동선 내러티브"**를 동적으로 생성 렌더링합니다.

### ③ Web Speech API Native TTS 낭독 엔진 ([GPanRadarPage](file:///d:/d:/000_UNSU/fo/src/pages/GPanRadarPage.tsx))
*   **브라우저 스피커 낭독**: `ON AIR` 버튼을 켜면, HTML5 표준 `speechSynthesis` API를 직접 호출해 기상 및 핫존 관제 텍스트 브리핑을 **한글 기사 음성(ko-KR, 0.92배속)**으로 실제 낭독하는 오디오 엔진을 구현했습니다.
*   **음소거 및 일시정지**: 음소거 아이콘 토글 혹은 ON AIR 해제(STANDBY 복귀) 시 `cancel()`을 호출하여 즉각 발화가 차단되도록 통제했습니다.

### ④ 영수증 OCR 리더보드 등극 및 허탕 피드백 ([RoadboarderPage](file:///d:/000_UNSU/fo/src/pages/RoadboarderPage.tsx))
*   **OCR 파일 업로드 시뮬레이션**: 영수증 OCR 버튼을 누르고 이미지를 선택하면 **"이미지 압축 및 OCR 분석 중"** 바(Bar)가 1.5초간 작동하며 판독 상태를 연출합니다.
*   **실시간 랭킹 추가**: 판독 성공 즉시 온보딩된 기사의 이름(마스킹 아이디)과 분석된 매출액(54,200원)을 리스트에 삽입하고, 기존 랭킹 목록을 순차적으로 밀어내어 **1위로 즉시 실시간 랭킹 등극**하는 극적 인터랙션을 가동했습니다.
*   **LangSmith 허탕 피드백**: 핫존 허탕 아이콘 클릭 시, 런타임 수집 데이터가 LangSmith Tracing Dataset에 안전하게 축적되었음을 모달로 알려줍니다.

### ⑤ 사업자 맞춤형 간이과세 부가세 계산기 ([AutopilotPage](file:///d:/000_UNSU/fo/src/pages/AutopilotPage.tsx))
*   **개인택시 간이과세 감안 환급 연산**: 온보딩 시 기사가 `개인택시 (PRIVATE)`를 선택했을 경우, 적격 매입세액 85.6만 원에 간이과세 신용카드 등 공제 가중치 50%를 적용하여 **₩ 428,000** 환급액을 도출합니다.
*   `모범/대형 (PREMIUM)`인 경우 일반과세자로 분류해 매입세액 100%인 **₩ 856,000** 환급 결과를 실시간 노출하며 코멘트를 분기 처리합니다.

로컬 서버 브라우저([Vite Dev App](http://localhost:5173))로 가셔서 아래 흐름으로 WOW 요소를 테스트해보세요!
1.  **G-PAN**: `ON AIR` 버튼을 눌러 스피커로 실제 한국어 인공지능 관제 멘트가 낭독되는지 확인합니다.
2.  **프로필**: 우측 상단 사람 아이콘을 눌러 생일과 홈택스 ID를 입력해 온보딩을 마칩니다.
3.  **루틴**: 메인 홈으로 가면 미등록 알림 배너가 사라지고 기사님의 생일에 맞춘 사주 행운 코멘트로 갱신된 것을 확인합니다.
4.  **정산**: 오토파일럿 탭에서 정산을 돌려 기사 종류(간이/일반)에 따라 환급 계산액과 문구가 동적으로 달라지는지 검증합니다.
5.  **로드보더**: 영수증 OCR 인증을 누르고 임의의 이미지를 첨부하여, 1.5초 로더 뒤 리더보드 1위로 실시간 랭킹 갱신이 일어나는지 모니터링합니다.
