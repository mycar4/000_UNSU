# 운수대통 프로젝트 핸드오버 (Handover)

## 📌 현재 작업 상태 (Current State)
1. **외부 API 동적 폴백(Fallback) 로직 완비**
   - KORAIL, METRO, AIRPORT 등 외부 API 키가 누락되거나 Mock 키인 경우, `DATA_GO_KR_API_KEY` (공공데이터포털 공통 키)로 자동 폴백되도록 `externalApi.ts` 로직 개선 완료.
2. **라이브 챗봇 (Gemini) 무한 Mock 응답 버그 수정**
   - 문제: Gemini API 호출 제한 시간(5초) 초과 시 에러를 삼키고 무조건 "사주 일진" 임시 응답만 뱉어내는 치명적 버그 존재.
   - 해결: `gemini.ts`의 Timeout을 30초(30000ms)로 연장하고, 에러 발생 시 정상적으로 `server.ts`로 에러를 던져 OpenAI(GPT) 폴백 엔진이 작동하도록 예외 처리 구조 수정 완료.
3. **카카오 맵 연동 상태**
   - 프론트엔드(`index.html` 및 `DarksidePage.tsx` 등)에 `VITE_KAKAO_MAP_API_KEY` 적용 및 UI 구현 완료.
   - 현재 SDK 로드 실패 에러 발생 중 (사유: 카카오 개발자 센터 도메인 미등록).
